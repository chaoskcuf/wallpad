코맥스 월패드 컨트롤러 - soecket 전용
===============================

commax-node
-----------
nodejs 를 이용하여 작동합니다.

commax-python
-------------
python3를 이용하여 작동합니다.

commax_mqtt2elfin
-----------------
socket 통신을 없애고 모두 mqtt를 사용합니다. python3를 이용하여 작동합니다.
